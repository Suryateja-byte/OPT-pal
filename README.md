# OPT-pal

OPT-pal is an Android app for F-1 students on OPT. It combines compliance tracking, secure document storage, document AI, reporting reminders, and onboarding flows that can extract setup details from I-20 and EAD documents.

## Tech Stack

- Android app with Kotlin and Jetpack Compose
- Firebase Auth, Firestore, Storage, App Check, Analytics, and Crashlytics
- Firebase Functions with TypeScript
- Google Cloud Vision and Vertex AI for document extraction

## Main Areas

- Auth and setup
- OPT dashboard and unemployment tracking
- Employment and reporting workflows
- Secure document vault, scan flows, and in-app viewer
- AI chat and document extraction
